# Pourya Gheitarani — Personal Website

Personal brand website for **Pourya Gheitarani**, Engineering & AI-Enabled Industrial Consultant.

**Live:** [pourya-website.vercel.app](https://pourya-website.vercel.app)

---

## Stack

- Single-page HTML with [Tailwind CSS](https://tailwindcss.com) via CDN
- Google Fonts: Space Grotesk · Inter · DM Serif Display
- No build step — open `index.html` directly in any browser

## Brand System

| Token | Hex | Role |
|---|---|---|
| Deep Engineering Blue | `#1A3A52` | Headings, nav, logo |
| Teal Intelligence | `#1E6B6B` | Subheadings, tags, icons |
| Industrial Copper | `#B87230` | CTAs, accents, highlights |
| Warm Off-White | `#F5F2EC` | Page background |
| Deep Charcoal | `#1E1E24` | Body text |
| Neutral Stone | `#E4E0D8` | Dividers, borders |

**Fonts:** Space Grotesk (headings/UI) · Inter (body) · DM Serif Display italic (hero tagline only)

## Sections

1. **Hero** — Name, tagline, role, stat strip
2. **Who I Am** — Personal brand statement and bio
3. **What I Bring** — Four core capabilities
4. **What I Stand For** — Five values
5. **A Defining Moment** — Field story quote
6. **Key Messages** — Five brand messages
7. **Contact** — Email + LinkedIn CTA

## Deployment

Deployed on [Vercel](https://vercel.com) with GitHub integration.  
Every push to `main` triggers an automatic redeploy.

```bash
# Manual redeploy (if needed)
npx vercel --prod
```

## Local Development

No setup required — just open `index.html` in a browser.
